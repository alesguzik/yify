# Yify

A Ruby wrapper for the [Yify torrenst API](http://yify-torrents.com/api/)

## Installation

Add this line to your application's Gemfile:

    gem 'yify'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install yify
